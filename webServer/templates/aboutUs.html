About Us

Why L3M?

L3M aims to promote democratization of generative AI models. We are a team of students at Stevens Institute of Technology working to create a product that will allow users to download and use open-source large language models with ease. We are a team of software engineering majors who want to improve accessibility and security of LLMs.

Project Description

We want to create a new generative AI model that can be stored as an application on personal computers. This would increase security for LLMs since there would be no broadband connection between a company server hosting the AI program and the computer the prompt was entered from. Our priorities for this project are to enable this AI to help users with everyday office tasks and train them to achieve different tasks.

Mission Statement

To create an easy installation package for a locally hosted large language model for personal and business use, to promote the security and democratization of LLM AI.
